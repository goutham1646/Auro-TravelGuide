# Puducherry Travel Guide Website

A beautiful, responsive website focused on fun places in Puducherry and the best food options near each place.

## Current Module

**Auroville Side Region** - Complete travel guide covering:
- Auroville (including Matrimandir viewing info)
- Serenity Beach
- Auroville Beach
- Nearby hidden spots tourists usually miss

## Files Structure

- `index.html` - Main HTML file with all content
- `styles.css` - Styling and responsive design
- `auroville-side-module.md` - Original markdown content (for reference)

## How to Use

1. Simply open `index.html` in any modern web browser
2. No server or installation required - works as a static website
3. The website is fully responsive and works on desktop, tablet, and mobile devices

## Features

- Modern, clean design with beautiful color scheme
- Fully responsive (mobile-friendly)
- Smooth scrolling navigation
- Well-organized sections with easy-to-read information
- Color-coded Do's & Don'ts sections
- Clear distinction between free and paid attractions
- Comprehensive food options with budget information
- Local tips and practical advice

You can easily customize:
- Colors in `styles.css` (CSS variables at the top)
- Content in `index.html`
- Add images by including `<img>` tags in the HTML
